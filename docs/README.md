## Documentation policy / 文档维护原则

- User-facing concepts should be documented in both English and Chinese.
- Security-sensitive decisions should explain the threat or failure mode.
- Phase plans should describe what is in scope and what is intentionally out of scope.

- 用户可见概念应尽量提供中英文文档。
- 安全相关决策需要说明威胁或失败模式。
- 阶段计划要写清楚范围内和刻意不做的内容。
