## 文档维护原则

- 根目录 README 负责项目介绍，保留中英文。
- `docs/` 下的架构、规格、设计稿说明和实施计划默认使用中文维护。
- 安全相关决策需要说明威胁或失败模式。
- Phase 实施计划要写清楚目标、非目标、任务拆分、测试和验收标准。
